How to setup TSAdminMQTT

Step 1

Firewall

Allow access.



Step 2

Tasmota settings

Enable MQTT



Step 3

MQTT settings

Host: Ip where TSAdminMQTT is running ({{ localIp }})

Adpater: Important under Topic the IP of the adapter must be entered!



Step 4

Setup TSAdminMQTT

Add rooms

Here rooms are added in this example Test Room 1 - 3.

Scan network for Tasmota devices

Click on Scan the process takes about 15 seconds

Assign adpter to a room

Here the network is searched for tasmota devices. All devices found are now listed. The devices found can now be assigned to a room via the dropdown menu.

Web credentials

At this point the web credentials for each adapter are stored. These are needed to get access to the adapter settings.he devices found can now be assigned to a room via the dropdown menu.

Finish setup

To save your configuration click on finish setup the application will restart.



Step 5

Use TSAdminMQTT

Here we are in the main menu, where we can select a room and there we will find the corresponding rooms. From here we can also access the Tasmota settings.
